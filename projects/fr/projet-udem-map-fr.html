Projet

Carte interactive du campus

Application web Next.js + MapLibre‑GL pour visualiser bâtiments, stationnements et points d’intérêt de l’UdeM.

Voir le site Code GitHub

Résumé

Carte interactive bilingue basée sur OpenStreetMap, avec couches pour bâtiments, parkings et POI du campus. Responsive et déployée sur Vercel.

Développement front‑end, intégration carto
Next.js, TypeScript, MapLibre‑GL, Vercel
2025

Lien rapide

Ouvrir l’application

Fonctionnalités

Couches

Affichage de polygones de bâtiments, parkings et POI avec styles dédiés.

Bilingue

Interface FR/EN et textes synchronisés avec le site.

Performance

Rendu WebGL fluide via MapLibre‑GL, déploiement Vercel.

Responsive

Mise en page adaptée mobile/desktop.

Open Data

Sources OSM et fichiers GeoJSON personnalisés.

Navigation

Zoom, déplacement, et contrôle de couches simples.

Détails techniques

Front‑end

Next.js avec TypeScript, composants réactifs, hooks pour MapLibre‑GL.

Cartographie

Styles MapLibre, couches GeoJSON, gestion des projections et tuiles.
